# 퍼스널 랜딩페이지 서비스 - 시작 가이드

## 🎉 축하합니다! 

**1단계 실행 우선순위 가이드**에 따라 성공적으로 완성되었습니다!

### ✅ 완성된 작업물

1. **메인 서비스 페이지** (`index.html`)
   - 서비스 소개 및 템플릿 선택
   - 요금 구분 및 제공 구성 안내

2. **템플릿 A - 개인 브랜딩형** (`templates/template-a.html`)
   - 퍼스널트레이너, 강사, 아티스트용
   - 인트로 + 대표문구 + 서비스 소개 + 신청폼

3. **템플릿 B - 비즈니스 소개형** (`templates/template-b.html`)
   - 디자이너, 작가, 포트폴리오용
   - 자기소개 + 포트폴리오 + SNS 링크 + 유튜브 임베드

## 🚀 지금 바로 확인하기

### 방법 1: 브라우저에서 직접 열기
```bash
# 메인 페이지
C:\cursor-project\personal-landing-service\index.html

# 템플릿 A
C:\cursor-project\personal-landing-service\templates\template-a.html

# 템플릿 B
C:\cursor-project\personal-landing-service\templates\template-b.html
```

### 방법 2: 로컬 서버 실행 (권장)
```bash
# Python이 설치된 경우
cd C:\cursor-project\personal-landing-service
python -m http.server 8000

# 그 후 브라우저에서 http://localhost:8000 접속
```
