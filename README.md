# 퍼스널 랜딩페이지 서비스

## 🎯 핵심 서비스 정의

**서비스명**: "2시간 완성 퍼스널 랜딩페이지"

**핵심 가치**: 클라이언트가 "당신이 뭘 해주는 사람인지" 바로 이해할 수 있는 개인 브랜딩 페이지

- ✅ PC + 모바일 반응형 디자인

## 📁 프로젝트 구조

```
personal-landing-service/
├── index.html              # 메인 서비스 소개 페이지
├── README.md               # 프로젝트 문서
├── styles/
│   ├── main.css           # 메인 페이지 스타일
│   ├── template-a.css     # 템플릿 A 스타일
│   └── template-b.css     # 템플릿 B 스타일
├── scripts/
│   ├── template-a.js      # 템플릿 A JavaScript
│   └── template-b.js      # 템플릿 B JavaScript
└── templates/
    ├── template-a.html    # 개인 브랜딩형 템플릿
    └── template-b.html    # 비즈니스 소개형 템플릿
```

## 🎯 템플릿 특징

### 템플릿 A - 개인 브랜딩형
- **대상**: 퍼스널트레이너, 강사, 아티스트, 컨설턴트
- **주요 섹션**:
  - 히어로 섹션 (강력한 메시지)
  - 서비스 소개 (3개 카드 레이아웃)
  - 고객 후기
  - 상담 신청 폼
- **특징**: 신뢰성과 전문성 강조

### 템플릿 B - 비즈니스 소개형
- **대상**: 디자이너, 작가, 프리랜서, 포트폴리오용
- **주요 섹션**:
  - 자기소개 (경험과 기술 스택)
  - 포트폴리오 갤러리
  - 유튜브 영상 임베드
  - SNS 링크
  - 프로젝트 문의 폼
- **특징**: 창의성과 작업물 중심

## 🛠️ 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: Flexbox, Grid, 반응형 디자인
- **JavaScript**: ES6+, 인터랙티브 기능
- **Font Awesome**: 아이콘
- **Google Fonts**: Noto Sans KR

## 📱 반응형 디자인

- **데스크톱**: 1200px 이상
- **태블릿**: 768px - 1199px
- **모바일**: 767px 이하

## 🚀 사용 방법

1. **메인 페이지 확인**: `index.html` 열기
2. **템플릿 선택**: 원하는 템플릿 클릭
3. **커스터마이징**: HTML/CSS 파일 수정
4. **배포**: 웹 서버에 업로드
